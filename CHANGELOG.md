# Changelog

All notable changes to `alpine-select-livewire` will be documented in this file.

## [1.0.0] - 2025-11-29

### Added
- Initial release
- Single select component (`<x-alpine-select::default>`)
- Multi-select component with chips (`<x-alpine-select::multiple>`)
- Searchable/filterable options support
- Clearable functionality for both variants
- Drag-and-drop reordering for multi-select
- Full Livewire wire:model integration with defer support
- Dark mode support out of the box
- TailwindCSS styling with zinc/blue color scheme
- Flexible options format (arrays, objects, mixed)
- Option normalization (value/label, id/nama, etc.)
- Custom event dispatching (model-updated)
- Inline and floating dropdown modes
- Select All/Clear All buttons for multi-select
- Comprehensive documentation and usage examples
- TailwindCSS v3 and v4 configuration guides
- View publishing for full customization
- Compatible with Laravel 11 and 12
- Compatible with Livewire 3.x
